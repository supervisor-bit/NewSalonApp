# Priority Aury

Aktualizováno: 6. 4. 2026

## P1 — Nejbližší priorita

### Sjednotit text našeptávače v mobilním builderu s desktopem
- zachovat **jednoduchý mobilní výběr**
- sjednotit **logiku názvu položky**
- nahradit méně hezké zobrazení typu v závorce (`Barva` / `Přeliv`) za čistší formát
- hlavní cíl: aby se položky **dobře poznaly a dobře ukládaly**

> Poznámka: není to akutní bug, ale UX priorita pro další klidný krok.

---

## P2 — Další doporučené kroky

1. **Dodavatel u dávkové příjemky**
   - dodavatel
   - číslo dokladu / faktury
   - poznámka

2. **Jednoduchý výdej / korekce skladu**
   - rozbití, vrácení, vyhození, ruční korekce

3. **Manažerský přehled skladu**
   - rychlý box / dashboard pro manažera
   - kolik položek je `Dochází`, `Objednáno`, `Rozdělané`
   - poslední příjem a co je potřeba řešit hned

4. **Minimum kusů pro produkty na doma**
   - jednoduché hlídání `Dochází`

5. **Export / tisk historie příjemek**
   - CSV nebo jednoduchý tisk dávky
